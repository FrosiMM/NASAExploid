# 🛸 NASA Space Explorer

Навчальний багатосторінковий сайт для тематичної роботи з інформатики (11 клас).  
Використовує офіційні відкриті API NASA для відображення реальних космічних даних.

## 📁 Структура файлів

```
index.html       — головна сторінка з банером та фото дня
mars.html        — галерея фото з поверхні Марсу
moon.html        — поточна фаза Місяця та астрономічні дані

stars.js         — анімований зоряний фон (генерує зірки через JS)
apod.js          — завантажує NASA APOD (фото астрономічного дня)
mars.js          — запити до Mars Rover Photos API
moon.js          — розрахунок фази Місяця + статичні факти
asteroids.js     — запити до NASA NeoWs (Near Earth Objects)
README.md        — цей файл
```

## 🌐 NASA API що використовуються

| API | Що робить | Документація |
|-----|-----------|--------------|
| **APOD** | Фото астрономічного дня | https://api.nasa.gov/#apod |
| **Mars Rover Photos** | Знімки з марсоходів Curiosity, Perseverance, Opportunity | https://api.nasa.gov/#mars-rover-photos |
| **NeoWs** | Астероїди поблизу Землі (Near Earth Objects) | https://api.nasa.gov/#NeoWS |

Ключ API: `DEMO_KEY` — безкоштовний, обмеження 30 запитів/годину.  
Для зняття ліміту отримай власний ключ на https://api.nasa.gov безкоштовно.

## 🧠 Як це працює

### Зоряний фон (`stars.js`)
Функція `createStars()` динамічно створює 180 HTML-елементів `<div class="star">`.  
Кожній зірці задаються випадкові: позиція (left/top у %), розмір, тривалість анімації та затримка.  
CSS-анімація `twinkle` змінює прозорість — зірки "мерехтять".

### Фаза Місяця (`moon.js`)
Розрахунок без API. Відомо, що 6 січня 2000 року о 18:14 UTC було новолуння.  
З тієї дати ділимо різницю в днях на синодичний місяць (29.53 дні).  
Дробова частина визначає поточний день циклу → індекс фази (0–7).

### Марс (`mars.js`)
Запит до `https://api.nasa.gov/mars-photos/api/v1/rovers/{rover}/photos?sol={sol}`.  
`Sol` — це марсіанський день (Sol 1 = перший день роботи марсохода).  
Відповідь повертає масив фото, ми показуємо перші 24.  
При кліку на фото відкривається лайтбокс (повноекранний перегляд).

### Астероїди (`asteroids.js`)
Запит до `https://api.nasa.gov/neo/rest/v1/feed?start_date=...&end_date=...`.  
NASA повертає об'єкти, згруповані по датах.  
Ми сортуємо за відстанню від Землі (найближчі — першими).  
`is_potentially_hazardous_asteroid` — булеве поле від NASA, що означає небезпеку.

## 🚀 Як запустити

Відкрий `index.html` у браузері. Для коректної роботи потрібне підключення до інтернету (запити до NASA API).  
Сервер не потрібен — все працює як статичний сайт.
